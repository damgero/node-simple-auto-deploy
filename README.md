# simple-auto-deploy
A simple utility to deploy a node app on another folder (used for development purposes)

## Installation

```bash
# Starting from 0.1.0 yarn is required
npm i -g yarn
npm i -g simple-auto-deploy
```

## Usage

```bash
simple-auto-deploy source target
```
